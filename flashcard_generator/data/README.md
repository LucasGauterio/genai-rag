# Sample Data Directory

This directory is for storing documents to be processed by the Flashcard Generator.

## Supported File Types

- **PDF** (.pdf) - Technical documents, textbooks, papers
- **Markdown** (.md) - Notes, documentation
- **Text** (.txt) - Plain text files

## Usage

Place your documents here and run:

```bash
python main.py ingest --input data/
python main.py generate --topic "Your topic"
```
